# Trader Behavior Analysis Using Bitcoin Fear & Greed Index

## Project Overview
This project analyzes how trader performance, risk-taking behavior, and trading activity change across different Bitcoin market sentiment regimes. The analysis combines Hyperliquid trading data with the Bitcoin Fear & Greed Index to understand how emotions influence trading outcomes.

The goal is to identify patterns in profitability, trade frequency, and risk exposure during periods of fear, neutrality, and greed.

---

## Datasets Used
- **Hyperliquid Trade Data**: Contains historical trade-level information such as position size, PnL, and timestamps.
- **Bitcoin Fear & Greed Index**: Represents overall market sentiment classified into fear, neutral, and greed regimes.

---

## Project Workflow
1. Data cleaning and preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Sentiment regime classification  
4. Performance and risk analysis by sentiment  
5. Visualization and insights generation

## Key Insights
- Trader profitability varies significantly across different sentiment regimes.
- Higher risk-taking behavior is observed during greed phases.
- Fear-driven markets show reduced trade frequency but more cautious positioning.
